[使い方]
Release版AudioConveter.exeにconvertしたいファイルを指定するとconvertが開始されます。
コンバート完了後には.aud形式のバイナリファイルと、コンバート後の各種情報が書かれたテキストファイルが出力されます。

使用例：コマンドラインからAudioConveter.exeにmp3, wav, ogg, flacのファイルパスを引数ににゅるよくして実行する

操作方法（一応）
1. コマンドラインでAudioConveter.exeがディレクトリに移動します
2. コマンドラインに./AudioConveter.exe "変換したいフォルダ名"で入力
3. そのファイルと同じディレクトリに.audが作成されます。
